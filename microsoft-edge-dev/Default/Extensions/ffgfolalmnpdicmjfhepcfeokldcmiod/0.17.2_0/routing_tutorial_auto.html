新增手动服务器选择功能

有时候，我们会需要通过特定区域的服务器来进行网络访问，比如上日本服务器看动漫

这时可以启用手动服务器选择功能。在服务器列表页面，点击选择手动指定

选择手动指定后，在服务器列表左侧

会出现如图所示的图标，提示可以手动调整服务器顺序

这时，可以通过拖动列表中的服务器

让中意的，排在第一个

用完记得要恢复成自动路由哦(省心省力)

这样就实现手动选择功能了
